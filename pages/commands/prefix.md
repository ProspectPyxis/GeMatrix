# prefix

Sets the bot prefix for this server.

*Note that you may ping the bot at any time to make it send the current prefix!*

### Alias(es)

*None available.*

### Usage
```
prefix [prefix]
```

## Parameters

### prefix

The new prefix to use. If left blank, will set the prefix to the bot's default prefix.

## Example(s)

- Set the server's prefix to `//`
  ```
  prefix //
  ```
