# help

Sends the list of commands, or details about the command if a parameter is passed.

### Alias(es)

*None available.*

### Usage
```
help [command]
```

## Parameters

### command

The command to get details for.

If left blank, the command will simply list every available command without details.

## Example(s)

- Get details for the `ping` command
  ```
  help ping
  ```
